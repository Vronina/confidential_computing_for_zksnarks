For this framework, benchmarks are available for the Groth16 and the Plonk prover.

# Groth16

## Setup:
To install all required prerequisites and copy all scripts into the created folder execute `chmod +x setup_gnark.sh` followed by `./setup_gnark.sh`.

## Run Benchmark:
To run the full benchmark, `cd sha_gnark/` and start the benchmark with `./run_benchmark.sh`.

## Run Benchmark:
